Guía de Configuración Inicial para Ubuntu 24.04

Despues de instalar Ubuntu estos son los primeros pasos que sigo para poner a punto mi sistema.

1. Actualización del Sistema

Primero, actualizamos la lista de paquetes y el sistema:

sudo apt update -y

sudo apt upgrade -y

sudo apt update -y

Estos comandos actualizan la lista de paquetes disponibles y luego actualizan todos los paquetes instalados a sus últimas versiones.

2. Instalación de Paquetes Esenciales

sudo apt install build-essential checkinstall

sudo apt install ubuntu-restricted-extras

sudo apt install software-properties-common

sudo apt install apt-show-versions

Estos comandos instalan herramientas de compilación, codecs multimedia, software para gestionar repositorios y una herramienta para mostrar versiones de paquetes.

3. Verificación de Actualizaciones

sudo apt upgrade -o APT::Get::Show-Upgraded=true

sudo apt-show-versions | grep upgradeable

Estos comandos muestran qué paquetes se actualizarán y cuáles tienen actualizaciones disponibles.

4. Configuración de SSH

sudo apt-get install openssh-server

sed -i 's/^PermitRootLogin.*/PermitRootLogin yes/' /etc/ssh/sshd_config

sed -i 's/^PubkeyAuthentication.*/PubkeyAuthentication yes/' /etc/ssh/sshd_config

sed -i 's/^PasswordAuthentication.*/PasswordAuthentication yes/' /etc/ssh/sshd_config

service ssh restart

Estos comandos instalan y configuran el servidor SSH, permitiendo el acceso remoto al sistema.

5. Limpieza Final del Sistema

sudo apt update -y

apt upgrade -y

sudo apt -f install

sudo apt autoremove

sudo apt -y autoclean

sudo apt -y clean

reboot

Estos comandos realizan una limpieza final del sistema, removiendo paquetes innecesarios y reiniciando el sistema.

6. Optimización de Energía (para laptops)

sudo apt install tlp tlp-rdw

sudo tlp start

Estos comandos instalan y activan TLP, una herramienta para optimizar la duración de la batería en laptops.

Nota: Algunos de estos comandos, especialmente los relacionados con la configuración de SSH, pueden tener implicaciones de seguridad. Asegúrate de entender completamente sus efectos antes de ejecutarlos en un sistema en producción.
